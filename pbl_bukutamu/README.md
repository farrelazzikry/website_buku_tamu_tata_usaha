PBL - Buku Tamu Politeknik Negeri Batam (XAMPP)
----------------------------------------------
Folder ini siap dicopy ke XAMPP `htdocs` (sebagai `pbl_bukutamu`).

Cara jalankan:
1. Copy folder `pbl_bukutamu` ke C:/xampp/htdocs/
2. Jalankan Apache & MySQL di XAMPP Control Panel.
3. Buka http://localhost/phpmyadmin — import file `database.sql` (di root folder proyek).
4. Buka http://localhost/pbl_bukutamu/ untuk akses form tamu.
   Admin: http://localhost/pbl_bukutamu/admin/login.php (user: admin / pass: admin123) — ganti password segera.
